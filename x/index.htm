Use this simple page as a URL placeholder or scratchpad for any bookmarklet.

As the creator of a bookmarklet, just make a link to: https://mobilemind.github.io/OpenInlets/x/#javascript:your-code-here

The hash string (named anchor) of the URL (the part after the '#') is simply ignored here, so you can use it for anything.

As an end-user, simply 'Add Bookmark' and save it. Then go back and edit the URL of the bookmark.

After saving the bookmark it must be edited as follows. Delete the https://mobilemind.github.io/OpenInlets/x/# part of the URL so that it begins with javascript: and continues with the rest of the original link. Finally, save the edited bookmark.
